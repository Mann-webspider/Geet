# Geet -- (Collaborative Music Streaming Platform)

### Overview
**Geet** is a collaborative music streaming mobile application designed to enable friends, couples, and groups to listen to music together, create shared playlists for events or trips, and collaborate in real time or asynchronously.

The platform focuses on low server load, smart caching, and user-driven music availability.


---
### Core Objectives

- Enable real-time and asynchronous collaborative music listening
- Support shared playlist creation for events and travel
- Provide individual silent listening using headphones
- Minimize server load using caching and eviction strategies
- Enable user-driven music availability requests
- Support lyrics integration for karaoke-style singing
